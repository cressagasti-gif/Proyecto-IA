# Proyecto-IA
Pequeño proyecto de programacion de IA

Este pequeño proyecto realizado a traves de Google Colab muestra un ejemplo de proyeccion y prediccion con inteligencia artificial, en el cual estan establecidos ciertos valores y variables que afectaran el resultado. El mismo es un ejemplo de prediccion de duración de neumaticos de coches de Formula 1, en cierto circuito, con ciertas paradas y ciertos compuestos
